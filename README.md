As part of the **React Intern Assignment**, I have developed a **pixel-perfect, front-end-only spreadsheet prototype** using **React 18**, **TypeScript**, and **Tailwind CSS**, replicating the provided Figma design. The project delivers a responsive, spreadsheet-like experience with essential interactions similar to Google Sheets/Excel.

---

## 🔧 Tech Stack

- ⚛️ **React 18** (Vite)
- 🟦 **TypeScript** (Strict Mode)
- 🎨 **Tailwind CSS**
- 📊 **react-table** for grid layout
- 🛠️ **ESLint + Prettier** for linting and formatting

---

## ✅ Features

- 🎯 **Pixel-perfect UI** implementation based on the Figma Design
- 📐 **Responsive spreadsheet-like table** with hover, active, and selected states
- 🖱️ All buttons and tabs trigger state changes or console logs — no dead UI
- 🧹 **Code Quality** ensured with:
  - `npm run lint` (ESLint + Prettier)
  - `npm run type-check` (TypeScript strict checks)
